# Machine Coding Solutions - Java

This repository contains my implementations for various **Machine Coding** (Low-Level Design) problems. These solutions are built with a focus on core software engineering principles, aimed at clearing rounds for top product-based companies.

## 🏗️ Project Structure

The repository is organized by problem folders. Each folder contains the complete source code, class structure, and logic for that specific challenge.

```text
.
├── Parking-Lot/          # Multi-level parking system implementation
├── [Future-Problem]/     # Placeholder for upcoming challenges
└── README.md             # Project overview
```

## 🛠️ Tech Stack & Requirements
Language: Java 17+

Build Tool: Maven / Gradle (where applicable)

Testing: JUnit 5 / AssertJ

Design Focus: SOLID Principles for maintainable architecture.

Design Patterns (Strategy, Factory, Singleton, Observer).

Thread Safety where concurrency is required.

## 📝 Design Philosophy
In every problem, I aim to cover the following "Machine Coding" evaluation criteria:

Correctness: Meeting all mandatory functional requirements.

Extensibility: Making it easy to add new features (e.g., adding a new vehicle type in Parking Lot).

Code Readability: Meaningful naming conventions and proper indentation.

Error Handling: Using custom exceptions and input validation.

Concurrency: Handling multiple requests simultaneously where applicable.
